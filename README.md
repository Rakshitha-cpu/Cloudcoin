## 📌 Overview

**CloudCoin** is a web-based platform that converts verified volunteer hours into digital currency. For every **1 hour** of confirmed social service, users earn **1 CloudCoin** — stored in a personal digital wallet and redeemable for rewards, certificates, or benefits.

> Volunteers contribute real value to society. CloudCoin makes sure that value is **measured, recognised, and rewarded**.

---

## ⚠️ Problem Statement

Traditional volunteering suffers from a fundamental gap:

| Issue | Impact |
|---|---|
| No recognition or incentives | Volunteers feel undervalued |
| Hours go untracked | No proof of contribution |
| Poor transparency | Low trust in the system |
| No scalable framework | NGOs can't manage rewards |

The result? Participation drops. Social impact weakens.

---

## 💡 Solution

CloudCoin introduces a simple but powerful model:

```
⏱ 1 Hour of Verified Service  =  ☁ 1 CloudCoin
```

This creates a **fair, transparent, and motivating** ecosystem for community service.

## ⚙️ Installation & Setup

```bash
# 1. Clone the repository
git clone https://github.com/your-username/cloudcoin.git

# 2. Navigate to the project folder
cd cloudcoin

# 3. Install Python dependencies
pip install -r requirements.txt

# 4. Run the Flask development server
python app.py
```

Then open your browser at:

```
http://127.0.0.1:5000/
```
